Rich Client
  • Today, the majority of web applications use the browser as a rich interactive application environment (Facebook, Twitter, Flickr, Google Docs etc.).
  • Great developments in client side technologies such as AJAX (JavaScript), Flash, Flex, Silverlight etc. are making browser applications (or web pages) look and behave almost like standard OS desktop applications (and sometimes nicer).
  • Although the client has been radically revolutionized, the server side architecture hasn't much! The application server stack is largely the same, and development of back-end supported rich client applications is still unnecessarily complex and expensive even when deploying on cloud computing resources.

As an example, this Grid is a rich client component, the data it contains is the result of an apstrata REST API call.
