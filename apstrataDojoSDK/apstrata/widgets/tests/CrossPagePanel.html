apstrata javascript SDK

The SDK is a versatile library provided to allow browser based Rich Internet Applications (RIA) to take advantage of the apstrata Rich Cloud Services.

The library is provide as open source under the GNU license agreement. It is written using the dojo toolkit, one of the most comprehensive open source javascript RIA platforms.

The SDK enables the development of fully featured data driven web browser based applications entirely in javascript.

The bundled reference applications demonstrate the abilities of the SDK and are completely free to use and modify.

  • Rich Client Presentation: Deck that explains how to use the Rich Cloud Services, with live examples and source code. open
  • Sample Apps: Collection of apps that take advantage of the Rich Client development model. open
  • Survey: a survey editor that can be used to generate embeddable surveys. open
