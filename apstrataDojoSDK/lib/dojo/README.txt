Unpack dojo clients here. The directory structure should look like:
    lib/dojo/<version.subversion.release>/dijit
    lib/dojo/<version.subversion.release>/dojo
    lib/dojo/<version.subversion.release>/dojox
    lib/dojo/<version.subversion.release>/util

e.g.:
    lib/dojo/1.2.3/dijit
    lib/dojo/1.2.3/dojo
    lib/dojo/1.2.3/dojox
    lib/dojo/1.2.3/util
