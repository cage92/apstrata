Unpack dojo clients here. The directory structure should look like:
    js/dojo/<version.subversion.release>/dijit
    js/dojo/<version.subversion.release>/dojo
    js/dojo/<version.subversion.release>/dojox
    js/dojo/<version.subversion.release>/util

e.g.:
    js/dojo/1.2.3/dijit
    js/dojo/1.2.3/dojo
    js/dojo/1.2.3/dojox
    js/dojo/1.2.3/util
