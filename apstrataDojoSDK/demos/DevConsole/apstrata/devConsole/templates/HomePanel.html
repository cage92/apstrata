apstrata Developer Console

This is where you will be defining the main apstrata objects to use in your application. It is the first step you should take before starting development. While this is something you could do entirely using the apstrata API with your own code, we are providing the console to save you time and make your task easier as you take your first steps into apstrata.

From the panel on the left, you can easily access the needed functions. You can add, delete, modify, export and import all object types. As a start, we suggest you create a store, define some users and take it from there.

And by the way, just so you know how cool apstrata is, this entire administration application is an RIA implemented in javascript and entirely running in the browser, it talks to apstrata's Rich Cloud services through the apstrata API, just like the applications you'll be writing.

Good luck with your next killer app project, we hope you enjoy developing with apstrata's APIs as much as we do! And if you get any issues or need support, let us know.
