apstrata Code Explorer

The Code Explorer contains a wealth of examples that illustrate how to write Rich Client applications (or RIA) using apstrata as back-end.



From the panel on the left, you can pick a mini-application to explore. The source code for these client-side javascript applications is provided as open source uner the GNU license at code.google.com/p/apstrata.





Good luck with your next killer app project, we hope you enjoy developing with apstrata's APIs as much as we do! And if you get any issues or need support, let us know.
