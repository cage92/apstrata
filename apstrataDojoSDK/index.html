apstrata javascript SDK

The SDK is a versatile library provided to allow browser based Rich Internet Applications (RIA) to take advantage of the apstrata Rich Cloud Services.

The library is open source under the GNU license agreement. It enables the development of fully featured data driven browser based applications entirely in javascript.

The bundled reference applications demonstrate the abilities of the SDK and are completely free to use and modify.

  • Rich Client Presentation: Deck that explains how to use the Rich Cloud Services, with live examples and source code. open
  • Developer Console: A complete developer console to administer your apstrata account. open
  • Code Explorer: A Collection of apps that take advantage of the Rich Client development model. open
