javascript SDK

the apstrata javascript SDK is a versatile library written using the dojo toolkit. The SDK enables the development of fully featured browser based database applications entirely in javascript.

The bundled reference applications demonstrate the abilities of the SDK and are completely free to use and modify.

  • Bliki: a versatile blog/wiki demo application. open
  • Survey: a survey editor that can be used to generate surveys that run in any webpage. open
