apstrata javascript SDK

The SDK is a versatile library to ebable mobile and desktop, browser based Rich Internet Applications (RIA) to take advantage of the apstrata Rich Cloud Services.

The library is open source under the GNU license agreement. It enables the development of fully featured data driven browser based applications entirely in javascript.

In addition to the data SDK, we provide a collection of useful widgets under the apstrata/ui package and a UI framework called Horizon under apstrata/horizon that makes it easy to implement data driven applications with readily available form, list and grid and other components.

The bundled reference applications demonstrate the abilities of the SDK and are completely free to use and modify.

  • CRUD demo: Demo code of the Create, Read, Update and Delete operations on the apstrata backend usign the apstrata SDK. open
  • Blue horizon: A demo of the Horizon UI framework that provides integration with apstrata backend through data forms, lists and grids. open
